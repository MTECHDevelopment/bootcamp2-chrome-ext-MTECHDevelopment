Esta página apresenta a Extensão Popup. Abaixo você poderá inserir capturas de tela (prints) mostrando o funcionamento: popup, opções, conteúdo, ícone, etc. A seção final mantém o código original exatamente como foi criado para referência.

Capturas de Tela

Como adicionar suas imagens

  1. Crie uma pasta docs/imgs (ou use outra) dentro do repositório.
  2. Faça commit das capturas de tela (PNG/JPG/WebP).
  3. Substitua cada figure.placeholder por: 
    <figure class="shot">
  <img src="./imgs/popup.png" alt="Popup da extensão" />
  <figcaption>Popup</figcaption>
</figure>
  4. Mantenha textos alternativos (atributo alt) descritivos para acessibilidade.
